# Indian Trading Analysis & Report Generator

Generate comprehensive analysis reports for Indian NSE/BSE stocks combining technical indicators, chart patterns, India VIX sentiment, and risk management.

## What This Skill Does

- Fetches real-time market data from Upstox / NSE / Yahoo Finance
- Calculates 9 technical indicators (RSI, MACD, Stochastic, Bollinger, Donchian, Williams %R, SMA, EMA, ATR)
- Detects 10 chart patterns (5 bullish, 5 bearish)
- Analyzes market sentiment using India VIX
- Generates BUY/SELL/HOLD recommendations with calibrated probabilities
- Computes ATR-based stop loss, take profit, and trailing stop levels
- Creates Markdown reports with all analysis data
- Exports as CSV or JSON for further analysis

## When to Use This Skill

Use this skill when the user requests:
- Technical analysis for Indian stocks (NSE/BSE)
- Investment recommendations for Nifty 50 or other Indian equities
- Market reports with technical indicators for Indian markets
- Portfolio analysis for Indian stock portfolios
- Comparative analysis across multiple Indian stocks

## Usage

### Single Stock Analysis

```bash
bun src/index.ts predict --ticker=RELIANCE.NS
```

### Multiple Stocks Comparison

```bash
bun src/index.ts predict --ticker=RELIANCE.NS,TCS.NS,INFY.NS,HDFCBANK.NS --format=json
```

### Portfolio Report

```bash
# Add stocks to portfolio first
bun src/index.ts predict --portfolio-action=add --portfolio-ticker=RELIANCE.NS
bun src/index.ts predict --portfolio-action=add --portfolio-ticker=TCS.NS

# Generate report for entire portfolio
bun src/index.ts predict --portfolio-action=report
```

### Detailed Fundamental Analysis

```bash
bun src/index.ts predict --fundamentals --portfolio-ticker=RELIANCE.NS
```

### Full Research (all data points)

```bash
# Run these in sequence for comprehensive analysis
bun src/index.ts predict --ticker=RELIANCE.NS
bun src/index.ts predict --fundamentals --portfolio-ticker=RELIANCE.NS
bun src/index.ts predict --earnings --portfolio-ticker=RELIANCE.NS
bun src/index.ts predict --dividends --portfolio-ticker=RELIANCE.NS
bun src/index.ts predict --news --portfolio-ticker=RELIANCE.NS
bun src/index.ts predict --options --portfolio-ticker=RELIANCE.NS
```

## Indian Market Specifics

- **Currency:** All prices in ₹ (Indian Rupee)
- **Market Cap:** Displayed in Crore / Lakh Crore (Indian notation)
- **Timezone:** IST (Asia/Kolkata) — market hours 9:15 AM to 3:30 PM
- **Sentiment:** India VIX (not US Fear & Greed Index)
- **Data Sources:** NSE Direct + Upstox API (Indian-specific) + Yahoo Finance (fallback)
- **Default Stocks:** 20 Nifty 50 blue chips

## Limitations

- Data freshness depends on source: NSE is real-time during market hours, Yahoo may lag 15-20 minutes
- Not financial advice — all outputs are for informational purposes only
- Options data works best via NSE Direct (Yahoo has sparse Indian options data)
- Indian quarterly results format differs from US GAAP — earnings data may be incomplete on Yahoo Finance
